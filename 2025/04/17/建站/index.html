⚠️ 值得注意的细节

  • GitHub Pages 的限制：

    • 每个 GitHub 账户只能有一个个人主页，仓库名必须为 <username>.github.io。
    • 项目页面的命名没有限制，可以有多个项目页面。

  • 个人主页与项目主页的区别：

    • 个人主页：地址为 https://<username>.github.io。
    • 项目主页：地址为 https://<username>.github.io/<repository>。

🔧 基本操作命令

  • 强制刷新浏览器缓存：Shift + F5
  • 清空缓存、部署、发送三部曲：
  • hexo clean：清除缓存文件。
  • hexo g：生成静态文件。
  • hexo d：部署到远程服务器（如 GitHub Pages）。
